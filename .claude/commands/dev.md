# Start Development Server

Start the Next.js development server for the beauty platform.

1. Check if any process is running on port 3000 and kill it if needed
2. Run `npm run dev` in background
3. Wait for server to be ready
4. Report the URL to access the application

The main site is at http://localhost:3000/ko
The admin login is at http://localhost:3000/ko/admin-login
